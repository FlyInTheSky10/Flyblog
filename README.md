### 简介

### 安装
```
npm install
npm link
```

### 使用

#### 构建到`/public`
```
flyblog generate
```
或者
```
flyblog g
```

#### 本地服务器运行
```
flyblog server
```
或者
```
flyblog s
```

#### 清理`/public`
```
flyblog clean
```
或者
```
flyblog c
```
